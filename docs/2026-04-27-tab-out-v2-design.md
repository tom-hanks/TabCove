# Tab Out 功能拓展设计文档

**日期:** 2026-04-27
**版本:** v2.0

---

## 概述

为 Tab Out Chrome 扩展新增 6 个功能，提升标签页管理效率。

---

## 功能 A: 闲置时间标注

### 功能描述
在每个标签页chip上显示上次访问时间，用相对时间标识。

### 显示格式
- 0-59 秒：刚刚
- 1-59 分钟：N 分钟前
- 1-23 小时：N 小时前
- 1 天及以上：N 天前

### 数据来源
使用 Chrome Tab API 的 `lastAccessed` 属性（毫秒时间戳）。

### UI 显示
在标签 chip 上增加一个小标签，显示在标题右侧或左侧：
```
🔵 GitHub ──────────── @owner/repo
             [3天前]        [保存] [×]
```

### 颜色区分
- 绿色 (#5a7a62): 1小时内
- 橙色 (#b8892e): 1-24小时
- 红色 (#b35a5a): 超过24小时

### 实现位置
- `app.js` 中的 `renderDomainCard()` 和 `buildOverflowChips()` 函数

## 功能 B: 一键创建 Chrome 原生分组

### 功能描述
点击域名卡片上的按钮，将该域名下的所有标签在浏览器中创建为原生标签组。

### API 使用
```javascript
chrome.tabs.group({ tabIds: [tabId1, tabId2, ...] }, (groupId) => {
  chrome.tabGroups.update(groupId, { title: 'GitHub' });
});
```

### UI 改动
在域名卡片的操作按钮区增加一个按钮：
```
[关闭全部 5 个标签] [创建标签组]
```

### 按钮样式
使用主色调按钮（accent-sage 绿色），表示这是一个"组织"类操作。

## 功能 C: 搜索与筛选

### 功能描述
在页面顶部增加搜索框，支持关键词搜索标签。

### UI 布局
```
[🔍 搜索标签页...                    ] [筛选▼]

  时间: 全部▼   域名: 全部▼   状态: 全部▼

  GitHub (12)  YouTube (5)  Gmail (3) ...
```

### 搜索功能
- 搜索范围：标签标题、URL、域名
- 实时搜索，输入即过滤
- 高亮匹配文字

### 筛选选项
1. **时间筛选** - 全部 / 刚刚（<5分钟）/ 今天 / 昨天 / 超过3天
2. **域名筛选** - 全部 / [列出所有域名]
3. **状态筛选** - 全部 / 有重复 / 已休眠

### 死链检测
使用 fetch 检测页面是否可访问。超过 3 次检测失败的标记为疑似死链（红色边框提示）。

## 功能 D: 标签页休眠

### 功能描述
自动休眠长时间未访问的标签页，释放内存。休眠后显示占位符，点击可快速恢复。

### 休眠规则
- **默认阈值:** 2 小时不活跃自动休眠（可自定义）
- **可配置:** 30分钟 / 1小时 / 2小时 / 6小时 / 1天
- **排除规则:** 固定标签页（pinned）不休眠

### 休眠实现
1. 检测标签 `lastAccessed` 时间
2. 超过阈值的标签，将其 URL 替换为占位页面
3. 占位页面显示原标题和 favicon，点击时恢复原 URL

### UI 显示
```
🔵 GitHub ──────────── @owner/repo
  [已休眠 · 点击恢复]        [保存] [×]
```

### 存储结构
```javascript
{
  suspended: [
    { id: "tabId123", url: "https://...", title: "...", suspendedAt: "..." }
  ]
}
```

### 设置配置
在 `config.local.js` 中配置：
```javascript
LOCAL_SUSPEND_THRESHOLD_HOURS = 2; // 默认2小时
```

## 功能 E: 历史与恢复

### 功能描述
记录最近关闭的标签页，支持快速恢复。

### 存储规则
- **最大保存:** 截止 2 天内的关闭记录
- **存储位置:** `chrome.storage.local` 中的 `recentlyClosed` 键
- **数据量上限:** 最多 500 条，超出后按时间淘汰

### 数据结构
```javascript
{
  recentlyClosed: [
    { url: "https://...", title: "...", closedAt: "ISO时间", tabId: 123 }
  ]
}
```

### UI 布局
新增一个"历史"面板，与"打开的标签"并列：
```
┌─ 打开的标签 ──┬─ 最近关闭 ──┐
│ GitHub (12)  │ 🔗 repo PR #456  │
│ YouTube (5)  │ 🔗 gist/abc      │
│ ...          │ 🔗 google.com     │
└──────────────┴─────────────────┘
```

### 恢复功能
点击历史记录中的项目：恢复该 URL 并从历史中移除。

## 功能 F: 快捷键支持

### 功能描述
添加键盘快捷键提升操作效率。

### 快捷键定义
| 按键 | 功能 |
|------|------|
| `/` | 聚焦搜索框 |
| `Esc` | 清除搜索 / 关闭弹窗 |
| `d` | 收藏当前标签（保存稍后读）|
| `x` | 关闭当前聚焦的标签 |
| `j` / `↓` | 下一个标签 |
| `k` / `↑` | 上一个标签 |

### 实现
使用 `document.addEventListener('keydown')` 监听键盘事件。
